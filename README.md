DMW Queue Management System
A comprehensive queue management system designed for the Department of Migrant Workers to streamline client service operations with real-time audio notifications and performance analytics.
Overview
The DMW Queue Management System helps manage client flow efficiently by providing a digital queue with audio calling capabilities, allowing counter staff to call clients systematically while administrators monitor service performance metrics.
Features
Core Functionality

Digital Queue Management - Automated client queue with ticket numbering
Audio Call System - Sound notifications when clients are called to service counters
Multi-Counter Support - Manage multiple service counters simultaneously
Real-Time Display - Client-facing display shows current queue status

Admin Dashboard

Performance Analytics - Track average clients served per day
Wait Time Monitoring - Monitor average queue wait times
Counter Efficiency - View average clients processed per counter
Queue Length Tracking - Real-time queue length statistics
Historical Reports - Access past performance data

Counter Interface

Call Next Client - Easy-to-use interface for calling the next client in queue
Audio Control - Trigger sound notifications for client calling
Client Status - View current serving and waiting clients

Technology Stack

Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Audio: JavaScript Web Audio API / HTML5 Audio
